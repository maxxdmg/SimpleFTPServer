# SimpleFTPSystem
A python implementation of a basic FTP client and server from scratch, without the python FTP library. 

##
Requires python 3

## Usage
### ftp_server.py
python3 ftp_server.py ip_address port

### ftp_client.py
python3 ftp_client target_address target_port

## client commands
  * help
  * retrieve file_name
  * list
  * quit
